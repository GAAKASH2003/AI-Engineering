# AI Engineering Learning Repository

This repository is a personal learning space for AI engineering. It documents my progress from foundational math concepts through the practical building blocks needed to understand and implement AI systems.

## What this repo is about

- Learning AI engineering from first principles
- Focusing on the mathematical foundations that power modern machine learning
- Building small, understandable modules as a study guide
- Keeping concepts simple, readable, and hands-on

## Current content

### `Math_foundations/`

A set of Python modules designed to help understand the core math used in AI engineering:

- `autograd.py` — tools for automatic differentiation and gradient computation
- `derivatives.py` — derivative rules and examples for learning calculus concepts
- `optimizers.py` — optimization algorithms used in training models
- `vectors.py` — vector math utilities for linear algebra and geometry

## Learning goals

- Strengthen understanding of calculus and linear algebra used in AI
- Learn how gradients are computed and applied in optimization
- Build reusable code that demonstrates fundamental AI engineering techniques
- Use this repo as a reference for future AI projects

## How to use this repo

1. Explore the `Math_foundations` modules.
2. Run and modify the examples to see how math maps to code.
3. Add new notebooks or modules as the learning path expands.
